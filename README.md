# Introduction to Morse Code

# What is Morse Code?
Morse code is a communication system that uses short signals (dots) and long signals (dashes) to represent letters and numbers. It allows messages to be sent using sound, light, or electrical signals.

# History of Morse Code
Morse code was invented by Samuel Morse in the early 1800s. It was first used with the telegraph to send messages over long distances. Before telephones were common, Morse code was one of the fastest ways to communicate. It was widely used in ships, the military, and emergency communication.

# How Morse Code Works
Each letter and number has a unique pattern made of:
- Dot (.) → short signal  
- Dash (-) → long signal  

Letters are separated by short pauses, while words are separated by longer pauses.

# Basic Examples:
